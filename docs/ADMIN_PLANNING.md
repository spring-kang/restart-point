# 어드민 사이트 기획서

## 개요

Re:Start Point 플랫폼의 운영자 전용 관리 사이트입니다.
시즌 생성, 수료 인증 승인, 사용자/팀 관리 등 운영에 필요한 기능을 제공합니다.

**관련 PRD 요구사항:** FR-01 (수료 인증), FR-03 (시즌 운영), FR-14 (운영자 대시보드)

---

## 구현 우선순위

### Phase 1: 핵심 운영 기능 (MVP)

1. **시즌 관리** - 시즌 생성/수정/조회
2. **수료 인증 관리** - 인증 요청 승인/반려

### Phase 2: 모니터링 기능

3. **대시보드** - 핵심 지표 시각화

### Phase 3: 확장 기능

4. **사용자 관리** - 사용자 조회/수정
5. **팀 관리** - 팀 조회/수정

### 2.3 시즌 API 매핑

| 기능 | HTTP Method | API Endpoint |
|------|-------------|--------------|
| 시즌 목록 조회 | GET | `/api/v1/admin/seasons` |
| 시즌 상세 조회 | GET | `/api/v1/admin/seasons/{seasonId}` |
| 시즌 생성 | POST | `/api/v1/admin/seasons` |
| 시즌 수정 | PUT | `/api/v1/admin/seasons/{seasonId}` |
| 시즌 상태 변경 | PATCH | `/api/v1/admin/seasons/{seasonId}/status` |
| 시즌 삭제 | DELETE | `/api/v1/admin/seasons/{seasonId}` |

### 2.4 시즌 상태 흐름

```
DRAFT → RECRUITING → TEAM_BUILDING → IN_PROGRESS → SUBMISSION → JUDGING → COMPLETED
  ↓         ↓            ↓              ↓            ↓           ↓
[생성]    [모집시작]   [팀빌딩시작]   [프로젝트시작]  [제출기간]  [심사기간]  [종료]
```

## 기술 스택

### 프론트엔드

- **프레임워크:** React + TypeScript
- **빌드 도구:** Vite
- **라우팅:** React Router v6
- **상태 관리:** Zustand (기존 프론트엔드와 동일)
- **HTTP 클라이언트:** Axios
- **스타일링:** Tailwind CSS

### 배포

- Vercel (기존 프론트엔드와 별도 프로젝트)
- URL: `admin.restart-point.com` (예정)

## 인증 및 권한

### 접근 권한

- ADMIN 역할을 가진 사용자만 접근 가능
- 일반 사용자는 403 Forbidden

### 인증 방식

- 기존 JWT 토큰 인증 사용
- 로그인 시 역할 확인
- ADMIN이 아닌 경우 접근 차단
