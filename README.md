# Re:Start Point

> 부트캠프 수료 이후의 성장을 다시 시작하는 AI 프로젝트 러닝 플랫폼

Re:Start Point는 부트캠프 수료 이후 자주 발생하는 학습 단절, 협업 기회 부족, 구조화된 피드백 부재를 해결하기 위해 만든 서비스입니다.  
수료생은 다시 팀을 이루어 시즌형 프로젝트에 참여하고, 운영자는 공모전/후속 교육을 체계적으로 운영하며, 심사자는 가이드를 기반으로 일관된 평가를 수행할 수 있습니다.

이 서비스는 단순한 프로젝트 모집 게시판이 아니라, **수료 인증 → 팀 빌딩 → 프로젝트 진행 → 전문가 심사 → 성장 리포트**까지 이어지는 후속 학습 루프를 하나의 플랫폼으로 연결합니다.

## Why Re:Start Point

부트캠프는 보통 "수료까지"는 촘촘하지만, "수료 이후"는 개인에게 맡겨집니다.

- 수료생은 학습 루틴이 끊기고, 협업 경험이 단절됩니다.
- 혼자 포트폴리오를 만들며 피드백 없이 성장 정체를 겪습니다.
- 운영자는 수료생 사후 관리와 공모전 운영에 많은 수작업을 투입합니다.
- 다음 기수 학습자는 좋은 프로젝트의 기준을 미리 배울 기회가 부족합니다.

Re:Start Point는 이 문제를 **AI 기반 팀 매칭, 프로젝트 코칭, 전문가 심사, 우수작 아카이브, 성장 리포트**로 해결합니다.

## Core Value

- **학습 지속성**: 수료 이후에도 프로젝트 중심의 성장 루프를 이어갑니다.
- **협업 복원**: 직무 기반 팀 구성으로 실전형 협업 경험을 다시 제공합니다.
- **피드백 구조화**: 심사 점수를 넘어 개인/팀 성장 리포트까지 연결합니다.
- **운영 효율화**: 시즌 운영, 인증, 심사, 리포트 발행을 하나의 흐름으로 관리합니다.

## Key Features

### 1. 수료 인증 기반 참여 제어
- 수료 인증을 완료한 사용자만 핵심 프로젝트 기능에 참여할 수 있습니다.
- 운영자는 인증 승인/반려를 관리할 수 있습니다.

### 2. 구조화된 프로필과 AI 팀 매칭
- 역할, 기술 스택, 관심 도메인, 협업 성향, 가능 시간 등을 바탕으로 프로필을 작성합니다.
- AI가 팀 추천과 멤버 추천을 제공해 팀 빌딩 비용을 낮춥니다.

### 3. 시즌형 공모전 운영
- 시즌 단위로 모집, 팀 빌딩, 프로젝트, 심사 일정을 운영합니다.
- 운영자는 시즌 상태와 규칙을 관리하고 전체 현황을 한눈에 확인할 수 있습니다.

### 4. 프로젝트 워크스페이스
- 팀 단위로 프로젝트를 생성하고 진행합니다.
- 주차별 체크포인트를 기록하며 진행 상황을 관리할 수 있습니다.
- 프로젝트 제출 전후 흐름이 상태 기반으로 관리됩니다.

### 5. AI 프로젝트 코칭
- 체크포인트 입력을 바탕으로 AI 피드백을 제공합니다.
- 리스크를 조기에 발견하고 다음 액션을 구체화할 수 있도록 돕습니다.

### 6. 전문가 심사 경험
- 전문가 심사위원이 루브릭 기반으로 프로젝트를 심사합니다.
- 시즌별 심사 비중은 전문가 평가 100%로 운영됩니다.
- 운영자는 전문가 심사 결과를 바탕으로 프로젝트 분석과 성장 피드백을 제공합니다.

### 7. AI 성장 리포트
- 프로젝트 종료 후 팀 리포트와 개인 리포트를 제공합니다.
- 단순 점수 공개가 아니라 강점, 보완점, 다음 성장 방향까지 제안합니다.

### 8. 우수작 아카이브
- 운영자는 심사 분석 화면에서 시즌 우수작을 직접 확정할 수 있습니다.
- 확정된 우수작은 메인 `우수작 보기` 페이지에서 시즌별로 공개됩니다.
- 다음 기수와 예비 참여자는 좋은 프로젝트의 기준을 실제 결과물로 확인할 수 있습니다.

### 9. 운영자 대시보드
- 시즌, 사용자, 심사, 리포트 현황을 관리자 화면에서 관리할 수 있습니다.
- 운영 리스크와 진행 현황을 빠르게 파악할 수 있습니다.

### 10. 커뮤니티와 알림
- 모집, Q&A, 쇼케이스, 공지 중심의 커뮤니티를 제공합니다.
- 댓글, 답글, 팀 지원/승인, 인증 결과 등 주요 이벤트를 알림으로 전달합니다.

## User Flow

### 수료생
1. 회원가입 후 수료 인증 요청
2. 프로필 작성
3. AI 추천을 참고해 팀에 지원하거나 직접 팀 생성
4. 시즌 프로젝트 진행 및 체크포인트 작성
5. 프로젝트 최종 제출
6. 심사 종료 후 성장 리포트 확인
7. 메인 `우수작 보기`에서 시즌 대표 프로젝트 확인

### 심사자
1. 심사 시즌 확인
2. 루브릭 기반 프로젝트 심사
3. 심사 완료 후 운영 분석 반영 확인

### 운영자
1. 시즌 생성 및 상태 관리
2. 사용자 인증/역할 관리
3. 대시보드로 운영 현황 모니터링
4. 심사 분석을 검토해 우수작 확정
5. 심사 및 리포트 운영

## What Makes It Different

### 1. 수료 이후에 집중한 서비스
많은 교육 서비스가 학습 과정 자체에 집중하는 반면, Re:Start Point는 수료 이후의 공백 구간을 제품의 핵심 문제로 다룹니다.

### 2. "프로젝트"와 "운영"을 함께 설계
학습자용 기능만 있는 것이 아니라, 실제 부트캠프 운영자가 공모전과 후속 교육을 관리할 수 있는 관리자 도구까지 포함합니다.

### 3. AI를 보조가 아닌 학습 루프에 연결
AI를 단순 챗봇으로 쓰지 않고, 팀 매칭, 프로젝트 코칭, 전문가 심사 분석, 성장 리포트 등 사용자의 전체 여정에 녹였습니다.

## Expected Impact

- 수료생의 학습 지속률 향상
- 프로젝트 기반 협업 경험 재연결
- 심사 결과의 성장 피드백화
- 운영자의 후속 교육 운영 효율화
- 전문가 심사 기반의 구조화된 피드백 제공

## For Competition

Re:Start Point는 "좋은 기능을 모아놓은 서비스"가 아니라, **부트캠프 이후 학습 단절이라는 실제 문제를 제품 구조로 해결한 플랫폼**입니다.

- 학습자에게는 다음 성장 기회를 제공하고
- 운영자에게는 운영 자동화와 데이터 기반 관리 도구를 제공하며
- 심사자에게는 일관된 전문가 평가 경험을 제공합니다.

즉, Re:Start Point는 수료 이후를 방치하지 않고, **성장이 다시 시작되는 지점**을 만드는 서비스입니다.
